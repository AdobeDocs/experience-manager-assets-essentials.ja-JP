# [!DNL Adobe Experience Manager Assets Essentials] について

[!DNL Adobe Experience Cloud]製品の一部として、Adobeは[!DNL Adobe Experience Manager Assets]という名前の堅牢なデジタルアセット管理（DAM）ソリューションを提供しています。 Adobe Experience Manager Assetsは、市場をリードするDAMです。 Assetsに加え、Adobe Experience Manager製品ラインの一部として、Adobeは現在、デジタルアセットを保存、管理、検索するための新しいSaaS ソリューションを提供しています。

[!DNL Adobe Experience Manager Assets Essentials]は、[!DNL Adobe Experience Manager Assets]の新しい軽量エディションです。[!DNL Assets Essentials] シンプルで一貫性のある使いやすいユーザーインターフェイスにより、統合されたアセット管理とコラボレーションを提供します。 これにより、より多くのクリエイティブ部門やマーケティング部門が、デジタルアセットを保存、発見、配信できるようになります。[!DNL Assets Essentials] 他のAdobe ソリューションと統合して、これらのソリューションでアセットを使いやすい方法を提供します。

[!DNL Assets Essentials]は、シンプルなインターフェイスと、デジタルアセットの効率的な検索と発見に重点を置き、チーム間でのアセット利用率、ガバナンス、ブランドの一貫性を確保します。 エンドユーザーエクスペリエンスは、スタンドアロンのweb ユーザーインターフェイスと、他のAdobe ソリューションに組み込まれたエクスペリエンスを介して実行できます。
